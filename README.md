# ComfyUI-Upscaling-Inpainting
Este repositorio incluye varios workflow de superresolución de imágenes con baja resolución, con modelos de sobreescalado como StableDiffusion, y multimodal, e implementaciones de baselines. En adición, viene un workflow aparte de Inpainting con el modelo de RealisticVision, la cual maneja la segmentación de máscaras con GroundingDINOSAMSegment.
